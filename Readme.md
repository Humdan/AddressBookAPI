# AddressBook API


BUILD INSTRUCTIONS:

navigate to project folder

install gorilla mux dependency package

	go get github.com/gorilla/mux

run the project:
	
	go build && ./yum

Open web browser to: http:localhost://8002

ENDPOINTS:
	GetPerson
	GetPeople
	CreatePerson
	DeletePerson

TEST INSTRUCTIONS:
use Postman to connect to the local server and test each endpoint
